# In-Bite

Basic mobile-first React + Tailwind starter for a food inspiration app powered by Gemini.

## Setup

1. Install Node.js so `npm` is available.
2. In `In-Bite`, run `npm install`.
3. Copy `.env.example` to `.env`.
4. Add your Gemini API key to `VITE_GEMINI_API_KEY`.
5. Run `npm run dev`.

## Stack

- React with Vite-style structure
- Tailwind CSS
- Gemini API via `fetch`

## Current MVP

- Mobile-first hero layout
- Meal idea form
- Gemini-backed suggestion generation
- Fallback starter cards when no API response is available
